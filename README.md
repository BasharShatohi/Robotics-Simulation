# Robotics Simulation - Cube Picking & Transport Robot

**Simulation of an autonomous robot that picks colored cubes and transports them along a predefined path to another robot in a virtual environment.**

**A robotics lab project simulating pick-and-place tasks with colored cubes, path following, and inter-robot coordination.**


## ✨ Features

- Virtual environment with divided areas and colored cube zones
- Autonomous robot navigation using sensors
- Cube detection, picking, and transport
- Path following with colored line markers
- Inter-robot communication and handoff
- Dynamic cube color configuration
- Real-time visualization

## 🛠 Technologies Used

- **Engine**: Unity3D
- **Language**: C#
- **Simulation**: Physics, Sensors, Pathfinding
- **Control**: Line following, Object manipulation

## 📋 Project Requirements Implemented

- Divided virtual environment with wall separator
- Colored cube zones (Red, Green, Blue, Yellow)
- Line paths for navigation
- Configurable color array on floor
- Cube picking and transport to second robot
- Sensor-based movement
